# vLLM Адаптер для VLMHyperBench

[English](README.md)

Стандартный адаптер для интеграции движка `vLLM` в платформу VLMHyperBench.

## Особенности

- Бесшовная интеграция с `vllm.AsyncLLMEngine`.
- Поддержка OpenAI-совместимых Chat Completions.
- Оптимизация для высокопроизводительного инференса.

## Установка

Требует предварительной установки `api-wrapper`.

```bash
pip install -e .
```

## Использование

Этот пакет предназначен для динамической загрузки ядром `api-wrapper`.
